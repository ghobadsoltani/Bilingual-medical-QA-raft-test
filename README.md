# AI Projects for Developing a Bilingual Medical Question Answering System

This repository contains a set of practice and experimental projects developed as part of the technical and research preparation for my thesis.

*Thesis Topic:*  
Design of a bilingual medical question answering system in the cardiovascular domain using cross-lingual reasoning based on the RAFT architecture.

## Repository Structure
```text
.
├── offline-medical-rag-demo
├── efficient-llm-adaptation
└── medical-ai-experiments

Each subproject has its own README file with detailed explanations, implementation notes, and project-specific documentation.

## Purpose

The purpose of this repository is to organize and document the projects, experiments, and implementation steps that contribute to the development of the main thesis project.

These subprojects support different parts of the overall system, including:

- medical question answering
- retrieval-augmented generation
- efficient adaptation of large language models
- experimental work in medical AI
- cross-lingual NLP

## Usage

Please navigate to each project folder and refer to its local `README.md` file for more information.

## Note

This repository is intended for research, learning, and development purposes.  
Its outputs should not be considered medical advice or a substitute for expert clinical judgment.
