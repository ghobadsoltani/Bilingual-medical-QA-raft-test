# medical-ai-experiments Repository

این پوشه شامل یک پروژه‌ی یادگیری و آزمایشی در مسیر `medical-ai-experiments/` است که روی موضوعات زیر تمرکز دارد:
- Retrieval-Augmented Generation (RAG)
- Embeddings و جستجوی معنایی
- Grounded question-answering در زمینه‌ی پزشکی
- استفاده از مدل‌های کوچک‌تر (7B) و اجرای محلی

## چطور از این ریپو استفاده کنید

```bash
cd medical-ai-experiments
python -m venv venv
venv\Scripts\activate    # Windows
pip install -r requirements.txt
jupyter notebook notebooks/
```

## ساختار

- `medical-ai-experiments/` : پروژه‌ی اصلی
- `medical-ai-experiments/README.md` : مستندات کامل پروژه
- `medical-ai-experiments/requirements.txt` : وابستگی‌ها
- `medical-ai-experiments/.gitignore` : قوانین نادیده‌گیری
- `medical-ai-experiments/notebooks/` : دفترچه‌های Jupyter
- `medical-ai-experiments/docs/` : مستندات فنی
- `medical-ai-experiments/data/` : نمونه داده‌ها
