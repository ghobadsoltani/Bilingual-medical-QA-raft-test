# offline-medical-rag-demo Repository

این پوشه شامل یک پروژه‌ی اصلی به نام `offline-medical-rag-demo/` است که یک نمونه کامل RAG آفلاین برای محتوای پزشکی آموزشگاهی را پیاده‌سازی می‌کند.

## محتویات

- `offline-medical-rag-demo/` : پروژه‌ی اصلی با کد پایتون، مستندات، داده‌های نمونه، و تست‌ها
- `docs/` : مستندات پروژه و راهنماهای فنی
- فایل‌های متنی خلاصه و چک‌لیست در ریشه

## نکته

برای استفاده یا انتشار در گیت‌هاب، بهتر است مستقیماً `offline-medical-rag-demo/` را به عنوان پروژه‌ی اصلی در نظر بگیرید. اگر می‌خواهید کل این پوشه را به عنوان یک ریپازیتوری قرار دهید، این README به عنوان معرفی کلی کفایت می‌کند.

## وضعیت

- پروژه‌ی `offline-medical-rag-demo/` آماده‌ی بررسی و اجرای اولیه است.
- یک `LICENSE` استاندارد MIT در داخل `offline-medical-rag-demo/` وجود دارد.
- یک `.gitignore` محلی در داخل پروژه‌ی اصلی وجود دارد، اما این README و `.gitignore` ریشه به نگهداری مرتب کمک می‌کند.
