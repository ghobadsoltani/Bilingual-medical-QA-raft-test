# Notebooks (Optional)

Space for exploratory Jupyter notebooks. Not required for the pipeline.

**Main entry points:**
- `python main.py --build-index` — Build FAISS index
- `python main.py --question "..."` — CLI queries
- `streamlit run app.py` — Interactive UI

**Notebook ideas:**
- Analyze retrieval scores for test questions
- Compare chunk size vs. retrieval quality
- Visualize embedding space (t-SNE)
- Manual evaluation of answer faithfulness to context
